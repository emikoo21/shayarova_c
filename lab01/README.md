# Лабораторная работа №1
## Типы данных, память и выражения в C

### Описание задачи:
В данной лабораторной работе необходимо изучить базовые типы данных в языке C, научиться работать с переменными, операторами ввода/вывода, а также понять связь переменных с памятью.

### Используемые технологии и библиотеки:
- Язык программирования: **C**
- Компилятор: **gcc** 
- Стандартная библиотека: **stdio.h**

### Инструкция по запуску:

#### Требования
- Компилятор GCC (MinGW для Windows, Xcode CLI для Mac, build-essential для Linux)
- Терминал/командная строка

#### Компиляция и запуск отдельных заданий
```bash
#### Задание 1
gcc task1.c -o task1
./task1

#### Задание 2
gcc task2.c -o task2
./task2

#### и так далее для каждого задания
```

#### Компиляция и запуск всех заданий сразу
```bash
#### Создайте скрипт run_all.sh (Linux/Mac) или run_all.bat (Windows)
#### Или просто компилируйте и запускайте каждое задание по очереди
```
