### Вопросы для самопроверки:

#### Что делает malloc
malloc выделяет заданное количество байт памяти. Память не инициализируется, значения остаются случайными. При ошибке возвращает NULL.

#### Чем отличается malloc от calloc
malloc принимает один аргумент и не инициализирует память. calloc принимает два аргумента и заполняет память нулями.

#### Что делает realloc
realloc изменяет размер ранее выделенного блока памяти. Может изменить адрес блока, сохраняет старые данные. При ошибке может вернуть NULL.

#### Что произойдет при утечке памяти
Выделенная память не освобождается и остается занятой до завершения программы.

#### Зачем нужен free
free освобождает ранее выделенную память, делая ее доступной для повторного использования.

#### Что произойдет при двойном освобождении памяти
Возникает ошибка, программа может аварийно завершиться.

#### Как проверить, что malloc вернул NULL
if (arr == NULL) { printf("Error\n"); }

#### В чем отличие статического и динамического массива
Статический массив имеет фиксированный размер, создается при запуске программы, автоматически освобождается. Динамический массив создается во время выполнения, размер можно менять, требует ручного управления памятью.
