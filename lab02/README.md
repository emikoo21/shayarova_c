### Вопросы для самопроверки:

#### Как объявить структуру в языке C
Структура объявляется с помощью ключевого слова struct, после которого указывается имя структуры и в фигурных скобках перечисляются поля с указанием их типов.

#### Что такое вложенные структуры и зачем они нужны
Это структуры, которые содержат поля, являющиеся другими структурами. Они нужны для группировки связанных данных, создания иерархических структур и повышения читаемости кода. 

#### Как передать структуру в функцию
Структуру можно передать в функцию двумя способами: по значению (копируется вся структура) и по указателю (передается адрес структуры). При передаче по значению используется имя структуры, при передаче по указателю используется оператор & при вызове и оператор -> внутри функции.

#### Можно ли создавать массивы структур
Да, можно. Массив структур создается так же, как массив любого другого типа. Доступ к элементам массива осуществляется через индексацию, а к полям - через оператор точки.

#### Как обратиться к элементу массива структур
Сначала указывается имя массива с индексом элемента в квадратных скобках, затем через оператор точки обращаются к нужному полю. Если используется указатель на структуру, то применяется оператор ->.

#### Можно ли использовать enum внутри структуры
Да, можно. Перечисление (enum) может быть использовано как тип поля внутри структуры. Это удобно для полей, которые могут принимать ограниченный набор значений.

#### Можно ли сравнивать структуры оператором ==
Нет, нельзя. Оператор == не определен для структур в языке C. Для сравнения структур необходимо сравнивать каждое поле по отдельности. Если структура содержит строки, то для их сравнения нужно использовать специальные функции, так как строки в C представлены массивами символов.
